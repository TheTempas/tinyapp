# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

### Getting Started

- Install all dependencies (`npm install`).
- Run the development web server (`node express_server`).
